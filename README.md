#### 001. 프로젝트 생성

<br>

```bash
$ npx create-next-app@latest --typescript
```

<br>

타입스크립트 기반 Next.js 프로젝트 생성 명령어를 치면 프로젝트명이 무엇이냐고 물어본다. 프로젝트명을 입력하고 나서 잠시 기다리면 Next.js 프로젝트 생성이 완료된다.

<br>

##### 001.1. development server 실행

```bash
$ npm run dev
```

<br>
